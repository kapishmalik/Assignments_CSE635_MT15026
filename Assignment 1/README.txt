Introduction

Developed Mobile application that has three buttons - start, stop, and send.
Pressing start will start collecting the time-stamped accelerometer data

Pressing stop will store the data in a CSV file

Send will send the data to the server

The server is developed using Sockets.

----------------------------------------------------------------------------
Required Information 

File Name saved at client side : AccelerometerDataClientCopy.csv 

File Name saved at Server side : AccelerometerDataServerCopy.csv

Port No used : 8097
